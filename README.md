# The Great Word Quest
A self-contained early reading game for a 5-year-old learner.
What is included
index.html
styles.css
app.js
There are no external libraries, downloads, build steps, tracking scripts, or server requirements. The game saves progress in the browser on the device.
Hosting on GitHub Pages
Create a new GitHub repository.
Upload the three game files from this folder.
In the repository settings, enable GitHub Pages.
Choose the branch and root folder that contain index.html.
Open the published GitHub Pages URL on the iPad.
iPad compatibility
The game is built as a simple static page for Safari on older iPads, including iPad Air 1 era devices. It avoids modules, build tools, external fonts, and network-only assets.
Sound and spoken words use browser features. If the iPad blocks audio until the first tap, press the Hear it or Sound on button once.
